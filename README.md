# 🛡️ Discord Security Bot

**Discord Security Bot для полной защиты серверов**

Полнофункциональный Discord-бот для комплексной защиты сервера. Более мощный, чем Wick, Beemo, SecurityBot, Guardian.

---

## ✨ Основные возможности

### 🔐 Система верификации
- **Капча-верификация** с случайными символами
- Автоматическая выдача ролей после верификации
- Логирование всех верификаций
- Персистентность после перезапуска (кэш капч, таймеры)
- Команда `!vfy` для отправки эмбеда верификации

### 🛡️ Защита от ботов
- **Whitelist система** для авторизованных ботов
- Автоматический кик/бан неавторизованных ботов
- Команда `!addl <bot_id>` для управления whitelist
- Логирование всех попыток добавления ботов

### 🚫 Антинюк
- Защита от массовых удалений каналов
- Защита от массовых банов
- Защита от массового создания ролей
- Автовосстановление удалённых элементов
- Защита от изменения прав ролей
- Защита от попыток забрать роль у бота
- Защита от попыток выдать себе админа
- Защита от удаления категорий
- Защита от изменения владельца сервера

### 🔥 Super Anti-Nuke
- **Защита от удалённых веток** (thread nuking) - 3+ удалений → бан
- **Защита от массового редактирования каналов** - массовое переименование → бан
- **Защита от изменения регионов сервера** - ограничение RTC-региона
- **Защита от отключения 2FA** у админов - логирование и предупреждение владельцу
- **Защита от массовой смены тем каналов** - быстрые изменения → захват

### 🔒 Super Anti-Webhook System
- Автоматическое удаление подозрительных вебхуков
- Проверка вебхук-ботов (>10 сообщений за 5 сек → удаление)
- Подмена вебхуков (отзеркаливание и замена на безопасные)

### 🔒 Полная защита сервера
- **Anti-Spam** система с автоматическим мутом
- **Anti-Flood** защита
- **Anti-Invite** фильтрация
- **Anti-Token Grabber** защита
- **Анти-фейк аккаунты** (проверка возраста аккаунта)
- **Защита от RAID** (массовый вход, автоматическое закрытие сервера)
- **Черный список слов** (мгновенный мут)
- **Анти-бот файлы** (защита от токен-ботов)
- **Авто-модерация сообщений:**
  - Капс
  - Ссылки
  - Спам
  - NSFW контент
  - Реклама
  - Фейковые приглашения

### 🔍 Дополнительные системы защиты
- **Anti-Ghost Ping** - защита от скрытых упоминаний
- **Anti-Webhook** - защита от вебхук-спама
- **Anti-MassMention** - защита от массовых упоминаний
- **Анти-Self-Bot** - обнаружение self-ботов
- **Проверка подозрительных никнеймов**
- **Защита от изменения имени сервера** с автовосстановлением
- **Защита от изменения иконки сервера** с автовосстановлением
- **Проверка активности администратора**
- **Лимиты на выдачу ролей** пользователями
- **Лимиты на удаление сообщений**
- **Удаление маскирующих ссылок** (disguised URL)
- **Anti-Spam в ЛС**

### 🧬 Protection Shadows (Тени-клоны)
- Создание невидимых резервных копий:
  - Копия всех ролей
  - Копия всех категорий
  - Копия всех прав
- Восстановление за 0.3 секунды при удалении

### 🔧 Auto-Heal System (Самолечение)
- Автовосстановление категорий/ролей/каналов
- Восстановление позиций ролей
- Восстановление permissions override
- Восстановление каналов с NSFW статусом

### 🔐 Lockdown System (Parental-Level)
- **FULL LOCKDOWN MODE:**
  - Блокировка создания каналов
  - Блокировка создания ролей
  - Запрет приглашений
  - Запрет переписок
- **RAID LOCKDOWN MODE:**
  - Активация при >10 пользователей/5 сек
  - Автоматическое временное закрытие сервера

### 👤 Trust System (Уровни доверия)
- Присвоение Trust Score (1-5) каждому участнику:
  - 1 = опасный
  - 2 = нейтральный
  - 3 = доверенный
  - 4 = старожил
  - 5 = ветеран
- Использование для автопроверок

### 🚨 Alarm System
- Уведомление всех модераторов при критических событиях
- Включение полной защиты
- Закрытие сервера при необходимости

### 📊 Security Dashboard
- Команда `/panel` для отображения:
  - Общее состояние защиты
  - Включённые модули
  - Статистика нук-попыток
  - Количество заблокированных атак
  - Трафик сообщений
  - Статистика рейдов
  - IP-агрегация (псевдо)

### 📝 Shadow Logs
- Отправка всех событий безопасности в ЛС владельцу
- Невидимые логи для полного контроля

### ⚙️ Модерация (Полная)
- **Автомод:**
  - Автоматический мут за спам
  - Счётчик предупреждений
  - Эскалация мутов при повторных нарушениях
  - Логи всех мутов

- **Slash-команды модерации:**
  - `/mute` - Заглушить пользователя
  - `/unmute` - Снять мут
  - `/kick` - Исключить пользователя
  - `/ban` - Забанить пользователя
  - `/warn` - Выдать предупреждение
  - `/unwarn` - Снять предупреждение
  - `/clear` - Очистить сообщения
  - `/slowmode` - Установить медленный режим
  - `/lock` - Заблокировать канал
  - `/unlock` - Разблокировать канал

### 🗂️ Управление чёрным списком
- `/blacklistadd` - Добавить пользователя в чёрный список
- `/blacklistcheck` - Проверить статус пользователя
- `/blacklistremove` - Удалить пользователя из чёрного списка
- Логирование всех действий в чёрном списке

### 🧰 Утилитарные команды
- `/avatar [user]` - Показать аватар пользователя
- `/userinfo [user]` - Полные данные пользователя
- `/serverinfo` - Информация о сервере
- `/channelinfo [channel]` - Информация о канале

### 📋 Команды аудита
- `/audit` - Показать последние 200 действий из Audit Log
  - Листание через серые стрелочки (⏮️ ◀️ ▶️ ⏭️)
  - Пагинация по 10 записей на страницу

### 💾 Команды бэкапа (только владелец)
- `/backup create` - Создать резервную копию сервера
- `/backup restore [backup_id]` - Восстановить структуру сервера
- `/backup list` - Показать доступные бэкапы

### 🔧 Команды владельца
- `/protectionoff` - Отключить абсолютно всю защиту
- `/protectionon` - Включить обратно абсолютно всю защиту
- `/restoreall` - Перепривязать всем роль unverified
- `/restore` - Восстановить сервер из бэкапа
- `/ownerlock` - Заблокировать всех модераторов (ограничение опасных прав)

### 🧪 Тестирование
- `/logtest` - Отправить тест-лог каждого типа для проверки

### 📢 Префиксные команды
- `!addl <bot_id>` - Добавить бота в whitelist (только админы)
- `!logs` - Показать информацию о системе логирования
- `!vfy` - Отправить эмбед верификации (только админы)

### 🔒 Anti-Permission Elevation
- Отмена попыток модераторов выдать себе больше прав
- Логирование всех попыток

### 💾 Server Backup-Snapshot Manager
- Периодическое сохранение структуры сервера (каждые X минут):
  - Роли
  - Каналы
  - Позиции
  - Permissions
- Сохранение в JSON в SQLite
- Автоматическое восстановление при подозрительных изменениях
- Ручное восстановление через `/backup restore`

## 📦 Установка

### 1. Клонирование репозитория
```bash
git clone https://github.com/yasimmy/Anti-Nuke.git
cd Anti-Nuke
```

### 2. Установка зависимостей
```bash
pip install -r requirements.txt
```

### 3. Настройка конфигурации

1. Откройте `config.json` и настройте:
   - **Токен бота** (`bot.token`)
   - **ID ролей** (`roles.verified`, `roles.unverified`, `roles.admin`)
   - **ID каналов** (`channels.verification`, `channels.verification_logs`, и т.д.)
   - **Параметры защиты** (включение/выключение модулей)
   - **Параметры анти-спама** и других систем
   - **ID сервера для синхронизации** (`bot.serversynh`)

2. Настройте статус бота:
   - `bot.status` - статус Discord (online, idle, dnd, invisible, offline)
   - `bot.activity_type` - тип активности (stream, playing, listening, watching, competing)
   - `bot.activity` - текст активности
   - `bot.stream_url` - URL стрима (для типа "stream")

### 4. Запуск бота
```bash
python main.py
```

## ⚙️ Настройка

Все настройки находятся в `config.json`. Вы можете:

### Основные настройки
- **Включить/выключить отдельные модули защиты**
- **Настроить параметры анти-спама** (количество сообщений, временное окно)
- **Настроить параметры верификации** (длина капчи, таймаут)
- **Добавить слова в черный список**
- **Настроить лимиты и таймауты**

### Настройка защиты
- `anti_bot.enabled` - включить/выключить защиту от ботов
- `anti_nuke.enabled` - включить/выключить антинюк
- `anti_spam.enabled` - включить/выключить анти-спам
- `anti_raid.enabled` - включить/выключить защиту от рейдов
- И многие другие...

### Настройка каналов логов
- `channels.verification_logs` - логи верификации
- `channels.join_logs` - логи присоединений
- `channels.security_logs` - логи безопасности
- `channels.moderation_logs` - логи модерации
- `channels.moderator_action_logs` - логи действий модераторов
- `channels.blacklist_logs` - логи чёрного списка

## 🔐 Требования к правам бота

Бот требует следующие права:
- ✅ **View Channels** - просмотр каналов
- ✅ **Send Messages** - отправка сообщений
- ✅ **Embed Links** - встраивание ссылок
- ✅ **Attach Files** - прикрепление файлов
- ✅ **Read Message History** - чтение истории сообщений
- ✅ **Manage Messages** - управление сообщениями
- ✅ **Manage Roles** - управление ролями
- ✅ **Manage Channels** - управление каналами
- ✅ **Kick Members** - исключение участников
- ✅ **Ban Members** - бан участников
- ✅ **View Audit Log** - просмотр журнала аудита
- ✅ **Administrator** (рекомендуется) - для полного доступа

## 📝 Логирование

Бот ведёт логирование всех важных событий:

- **Верификация** - успешные верификации
- **Присоединения** - новые участники
- **Безопасность** - попытки атак, нук-попытки
- **Модерация** - муты, баны, кики
- **Действия модераторов** - кто что сделал
- **Чёрный список** - добавления/удаления

Все логи отправляются в соответствующие каналы, настроенные в `config.json`.
